## Instrucciones:

1) Agregar la siguiente dependencia a tu composer.json

```json
"require-dev": {
    ...
    "composer require eltresm/hello-composer": "dev-main"
},
```

2) Actualizar tus dependencias
```bash
composer update
```

3) Ver carpeta de example

```bash
php index.php
```
